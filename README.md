# JK Lakshmi Vehicle Requisition Management System

A web-based Vehicle Requisition Management System developed for JK Lakshmi Cement to digitize and automate the vehicle booking and approval process across departments.

## Features

* Employee Login & Authentication
* Vehicle Requisition Submission
* Multi-Level Approval Workflow
* Vehicle Allocation Management
* Transport Desk Operations
* Request Tracking & Status Monitoring
* Analytics & Reporting Dashboard
* MySQL Database Integration
* Multi-User Access over LAN

## Technology Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MySQL
* Deployment: Local Network / IIS Compatible

## Objective

The system replaces manual vehicle requisition processes with a centralized digital platform, improving efficiency, transparency, and tracking of transportation requests within the organization.
